In Lacan's mathemes of the psychoanalytic discourses, each matheme contains four symbols: The master signifier (S1), knowledge (S2), the barred subject (S̷) and the object cause of desire (a).

Fill in the grid so that each row, column, and 2x2 square is a complete matheme, containing each symbol once.
